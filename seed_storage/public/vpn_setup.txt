============================================
  IntraCore VPN 설정 매뉴얼
  Version 1.3 | 2026-02-01
============================================

1. VPN 클라이언트 다운로드
   - 사내 소프트웨어 배포 시스템에서 IntraCore VPN Client를 다운로드합니다.

2. 설정 파일 적용
   - 인프라팀에서 발급한 .ovpn 파일을 클라이언트에 import 합니다.
   - 서버: vpn.intracore.local:1194
   - 프로토콜: UDP

3. 인증
   - 사내 계정과 동일한 자격 증명을 사용합니다.
   - MFA가 활성화된 경우 OTP 코드를 추가로 입력합니다.

4. 문제 해결
   - 연결 실패 시 인프라팀(내선 5500)에 문의하세요.
   - 방화벽 규칙 문제는 헬프데스크 티켓을 생성해 주세요.
